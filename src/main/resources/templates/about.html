About Smart Contact Manager

The Smart Contact Manager is a web application designed to help users efficiently manage their contacts. It provides a smarter and more organized way to store, access, and manage contact information, making it easier to keep track of personal or professional relationships.

Key Features:

  • User Registration & Authentication: Secure user registration with encrypted passwords and role-based access.
  • Role-Based Access Control: Admins have control over user management and other administrative tasks.
  • Contact Management: Add, edit, delete, and view contacts with detailed information.
  • Search Functionality: Quickly find contacts with a dynamic search feature.
  • Responsive Design: Accessible on various devices including desktops, tablets, and smartphones.
  • Secure Data Handling: Employs Spring Security with encryption and secure session management.
  • User-Friendly Interface: Intuitive navigation with easy access to different sections like Home, About, Login, and Sign-Up.
  • Scalability: Built with Spring Boot, scalable, and easy to deploy.

Use Cases:

  • Personal Use: For individuals managing personal contacts.
  • Business Use: For professionals keeping track of client or colleague information.
  • Organizational Use: For teams or small businesses needing a centralized contact management system.

Technical Stack:

  • Backend: Spring Boot, Hibernate, MySQL
  • Frontend: Thymeleaf, HTML, CSS, JavaScript, Bootstrap
  • Security: Spring Security with BCrypt encryption
  • Deployment: Compatible with any server supporting Java applications

The Smart Contact Manager aims to simplify contact management while providing robust security and a user-friendly experience.
